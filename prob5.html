HTM5 Features

Headings

HTML provides six levels of headings, from <h1> to <h6>, to structure content hierarchically. These tags help organize the page and provide semantic meaning to different sections.

Lists

HTML supports ordered lists (<ol>) and unordered lists (<ul>). Ordered lists display items with numbers or letters, while unordered lists use bullets. List items are defined with <li> tags.

Hyperlinks

Hyperlinks are created with the <a> tag. They allow you to link to other web pages, resources, or files. You can specify the destination URL using the href attribute.
